# Morpy

Sushi-mockeurs is a **Java J2EE** Application that runs on top of **Spring** and **Maven**.

## How to run

Go to the root of this project and run:

```sh
$ mvn spring-boot:run
```

To run the specs:

```sh
$ mvn test
```
